# 데이터

## 데이터의 수행
1. Select or Collect (자료수집)
2. Preprocessing (전처리)
3. Transform (변환)
4. Model (모델)
5. Evaluation (평가)

## 통계학과 빅데이터
|통계학|빅데이터|
|-|-|
|표본조사 (Sample)|전수조사 (Census)|
|질|양|
|사전처리|사후처리|
|인과 (Casualty)|상관 (Correlation)|
