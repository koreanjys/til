프로젝트를 만들었을때 글로벌환경에 pip list를 해보면 너무 많은게 깔려있어서

내가 뭘 사용해서 프로젝트를 만들었는지 잘 모른다.

다른 사람들과 협업을 하거나 , 가상환경에서 서버를 구동할때 딱 필요한거만 설치하기 위해서

venv가 필요하다.

명령어를 기억하자

1. python -m venv <env name>    초록색 네모칸(폴더 venv) 생성

2. source venv/Scripts/activate      MTV폴더가 글로벌환경을 가리키는걸 venv를 가리키게 하는 명령어
   (deactivate) => 이건 다시 글로벌 환경을 가리키게 하는 명령어

3. 이제 pip install 을 하면 노란 폴더에 프로젝트마다 독립적으로 설치가 된다.

4. git 사용폴더에 만들었으면  .gitignore 를 만들어서 구글에 gitignore.io 에 가서 설치된 패키지를 검색창에 입력해서 (여러개 가능) 생성한다음
  복붙 하면된다.
